#### 排序算法

##### 冒泡排序

从第一个元素开始

第一个元素和第二做比较，第一个元素和第二个元素比较，一直比到最后一个元素

第二个元素和第三个比较，第二个和第四个比较，一直比到最后一个

```python
li = [1, 2, 34, 2334, 4, 5, 423, 3, 2, 2, 2, 243, 4, 5, 6, 5, 4, 6, 7, 8, 5, 3, ]

for n in range(len(li)):
    for i in range(len(li) - n - 1):   # 这里是为了控制还需要比较的元素的个数
        if li[i] > li[i + 1]:
            li[i], li[i + 1] = li[i + 1], li[i]

print(li)
```

#### 希尔排序


```python
# 希尔排序
ls = [49,38,65,97,76,13,27,49]
length = len(ls)
# 增量
j = length // 2
while j >= 1:
    for i in range(j,len(ls)):
        while i-j >= 0 and ls[i] < ls[i-j]:
            ls[i],ls[i-j] = ls[i-j], ls[i]
            i -= j
    j = j // 2
print(ls)
```

#### 归并排序

```python
# coding: utf-8
li = [1, 23, 10, 4, 7, 11, 15]

# 归并排序
def merge(left, right):
    i, j = 0, 0
    result = []
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    result += left[i:]
    result += right[j:]
    return result

def merge_sort(lists):
    if len(lists) <= 1:
        return lists
    num = len(lists) // 2
    left = merge_sort(lists[:num])
    right = merge_sort(lists[num:])
    return merge(left, right)

b = merge_sort(li)

print(li)
print(b)

```
