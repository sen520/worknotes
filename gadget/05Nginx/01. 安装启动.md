#### Nginx 的作用是什么

1.  HTTP 服务器处理静态资源 
2. 正向代理
3. 反向代理 + 负载均衡
4. 动静分离



- windows 

解压运行 nginx.exe 即可

- Linux 也一样

./configure --prefix=/opt/nginx

make

make install

centos 7 关闭防火墙

systemctl stop firewalld.service

启动 nginx 

./nginx



修改配置文件 记得把 nobody 改成当前的用户



- 正向代理 （一般需要在客户端配置）

代理服务器代理的是客户端，即服务端不知道客户端的地址，==客户端知道服务器的地址==

- 反向代理

代理服务器代理的是服务器，即客户端不知道服务器的真实地址，服务器知道客户端的真实地址
